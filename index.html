Infra

Supabase + Vercel

  1. Crie um projeto no Supabase e rode o SQL de supabase/schema.sql.
  2. Cadastre no Vercel as envs SUPABASE_URL, SUPABASE_ANON_KEY e ADMIN_EMAILS.
  3. Faça deploy desta pasta no Vercel. A pagina usa /api/config para ler as envs em runtime.
